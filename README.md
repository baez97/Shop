# Sports shop
Sports shop is a web application project for Web & Services Engineering subject of the Computer Science Degree at UCLM. It consist on a e-commerce web application for buying Sport stuff.

The technologies used are:
* MongoDB & Mongoose for persistence
* Passport.js for authentication
* NodeJS for the APIRest
* Handlebars for template rendering
* Bootstrap for the responsive design
* HTML, CSS and JavaScript
